# Fibroblast Control Panel

Interactive tool: 3D fibroblast state map, transcription-factor drivers, per-element
evolutionary inspector, motif-turnover phylogeny, and predicted 3D architecture across
mammals. Static site — open `index.html` or host anywhere.

The "Ask Claude" copilot needs a small backend with an API key (optional); without it the
rest of the tool works fully.
